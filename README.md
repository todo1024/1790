课程介绍：

云原生已是毋庸置疑的技术发展趋势之一。PaaS作为云原生体系的核心架构层，正被越来越多的公司应用，PaaS工程师也成为企业招聘热门资源。Go开发者，正是PaaS工程师的主要人才来源。本课程将带领大家，结合Go微服务打造PaaS平台的核心业务（包括Pod，service，deplyment，Ingress，存储，监控，中间件，镜像市场等），帮助Go工程师探索PaaS开发，挖掘职业新可能。
